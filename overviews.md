## **Rythmos® Anchor Type C**

The **Rythmos® Anchor Type C** is a battery-powered mesh node installed throughout the community to support indoor resident positioning and alert routing. It helps form a self-healing Rythmos® mesh network that collects location and event data from wearables and passes that information toward the Gateway and Care Manager platform. Compact and fixed-position, the Anchor Type C is designed for corridors, rooms, exits, and shared spaces where reliable mesh coverage and location accuracy are essential.

The **Rythmos® Gateway** is the central connectivity hub for the Rythmos® ecosystem, aggregating data from wearables, anchors, sensors, and access devices across the community. With wired LAN, Wi-Fi, and mesh connectivity, the Gateway connects the on-site device network to cloud-based care workflows and centralized system management.
