# Rythmos Data Sheets Generator 📄

This project is designed for the rapid generation of technical Data Sheets for Rythmos devices. It allows you to create 25+ unique pages using a single professional template and data stored in JSON format.

## How to Run

### 1. Install Necessary Tools
To run this project, you need **Node.js** installed on your computer.
*   Download and install it from the official website: [nodejs.org](https://nodejs.org/) (choose the **LTS** version).

### 2. Preview the Template
To see what the page looks like currently:
1.  Open your Terminal (or Command Prompt).
2.  Navigate to the project folder.
3.  Type the following command:
    ```bash
    npm start
    ```
4.  A link will appear in the terminal (e.g., `http://localhost:3000`). Copy and paste it into your browser.
5. Choose "template.html" in the browser.
