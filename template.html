Overview

The Rythmos® Anchor Type F is a USB-C powered ambient sensing mesh network node designed for healthcare and senior living environments. It contributes real-time resident location data and passes through alert information while simultaneously collecting environmental data — including temperature, humidity, barometric pressure, ambient light, and sound level — all through a single compact, wallmounted unit that integrates directly into the Rythmos® mesh infrastructure

System Topology — Anchor Deployment

Deployment tip: Install anchors throughout corridors, common areas, and near exits. Anchor density directly affects location accuracy — Intrex recommends placement every 30 ft in typical senior living corridors. Install at least 1 anchor in every room, and multiple in high-occupancy areas such as dining rooms. Anchors must remain fixed post-calibration; repositioning requires recalibration via the Rythmos® Care Manager portal.
